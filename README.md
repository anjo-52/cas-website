# 🎓 CAS Malampuzha College Website

A responsive college website built using **React.js**.  
This project showcases a modern web interface for a college, including key sections such as Home, About, Courses, and Contact.

---

## 📌 Project Overview

This project is a **college website interface** designed to provide information about CAS Malampuzha, including available courses and contact details.

The website is fully responsive and works across:

- Desktop 💻  
- Tablet 📱  
- Mobile 📲  

---

## 🚀 Features

- Responsive Navigation Bar
- Hero Section with College Image
- About Section
- Courses Listing
- Contact Form
- Footer Section
- Mobile-Friendly Design
- Clean UI Layout

---

## 🛠️ Built With

- React.js
- HTML5
- CSS3
- JavaScript
- Vite (Development Tool)
